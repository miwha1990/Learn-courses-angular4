
Common information:
Success code: always 200
Headers: 'Content-Type: application/json'
Possible Failure codes: 400, 403, 404, 409, 429, 500, 503, 504
In case of any error response contains json body, which has the follwing format:
{"message": "error message text here"}

GET https://api.dtsfitnesseducation.com/v1/classes?category_id=&course_id=&location_id=
    Parameters:
        category - optional, string, id of any category from /categories API response
course - optional, integer, id of any course from /courses API response
location - optional, integer, id of any location from /locations API response
